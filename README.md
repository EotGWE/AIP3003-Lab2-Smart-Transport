# AIP3003-Lab2-Smart-Transport

How to Run the Dashboard Locally:
Extract all files (dashboard.py and smart_transport_dataset.csv) into the same folder.
Ensure you have the required Python libraries installed in your environment. If not, install them using the following command:
pip install streamlit pandas plotly
Open your terminal or command prompt, navigate to the folder containing these files, and execute the following command:
streamlit run dashboard.py
The interactive dashboard will automatically open in your default web browser (typically at http://localhost:8501).
